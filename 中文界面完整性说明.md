# 📊 中文界面完整性说明

## 概述

本项目已尽最大努力实现中文界面支持。由于不同监控工具的技术限制，中文化程度有所不同。

## 中文化程度对比

| 服务 | 界面语言 | 中文化程度 | 说明 |
|------|---------|-----------|------|
| **Grafana** | ✅ 完全中文 | 100% | 官方支持多语言，已配置为中文 |
| **Prometheus** | ❌ 英文 | 50% | 官方不支持多语言，但告警规则和配置已中文化 |
| **Alertmanager** | ❌ 英文 | 40% | 官方不支持多语言，但配置文件已中文化 |
| **Kibana** | ✅ 可选中文 | 90% | 官方支持多语言，需手动设置 |
| **Jaeger** | ❌ 英文 | 0% | 官方不支持多语言 |
| **Ceph Dashboard** | ⚠️ 部分中文 | 60% | 取决于 Ceph 版本 |

## 详细说明

### 1. Grafana（推荐主要使用）⭐⭐⭐⭐⭐

**中文化程度：100%**

✅ **完全中文化：**
- 界面语言：简体中文
- 菜单和按钮：中文
- Dashboard 标题：中文
- 面板标题：中文
- 图例标签：中文
- 数据源配置：中文注释
- 错误提示：中文

**配置方式：**
```yaml

  - GF_DEFAULT_LOCALE=zh-CN
```

**使用建议：** ⭐⭐⭐⭐⭐
- 日常监控的主要工具
- 完全中文界面，易于理解
- 强大的可视化功能
- 推荐使用频率：每天

### 2. Prometheus（调试工具）⭐⭐⭐

**中文化程度：50%**

❌ **无法中文化：**
- Web UI 界面（硬编码为英文）
- 菜单和按钮
- 表格标题
- 状态标签

✅ **已中文化：**
- 告警规则描述（annotations）
- 告警摘要和详情

**使用建议：** ⭐⭐⭐

- 参考 `Prometheus界面语言说明.md`

### 4. Kibana（日志分析）⭐⭐⭐⭐

**中文化程度：90%**

✅ **可以中文化：**
- 界面语言：简体中文
- 菜单和按钮：中文
- 设置页面：中文
- 错误提示：中文

⚠️ **需要手动设置：**
1. 访问 Kibana
2. Stack Management → Advanced Settings
3. 搜索 "locale"
4. 设置为 "zh-CN"

**配置方式：**
```yaml
environment:
  - I18N_LOCALE=zh-CN
```

**使用建议：** ⭐⭐⭐⭐
- 日志分析和搜索
- 完整的中文界面
- 推荐使用频率：经常（如果使用 ELK）
